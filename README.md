# 6副牌带鹰够级｜手机网页游戏

这是一个可以部署到服务器、让 6 个人用手机浏览器一起玩的够级网页游戏原型。

## 已包含

- 6 人房间
- 房间码加入
- 手机浏览器邀请 / 复制链接
- WebSocket 实时同步
- 6 副扑克牌 + 6 张“鹰”
- 鹰牌最大，可作为挂牌加入同点数组合
- 0/2/4 vs 1/3/5 联邦
- 触控选牌、出牌、过牌
- 走科、末科、落贡提示
- 移动端自适应布局

## 当前规则说明

公开资料中，够级是 6 人分两方、间隔而坐的山东青岛起源扑克游戏；部分平台有“6副牌带鹰”玩法。带鹰各地细节可能不同，本版本为了先能开房联机，把“鹰”作为最大特殊牌处理：

- 牌堆：6 副标准扑克牌 + 6 张鹰，共 330 张。
- 每人 55 张。
- 牌序：鹰 > 大王 > 小王 > 2 > A > K > Q > J > 10 > 9 > 8 > 7 > 6 > 5 > 4 > 3。
- 出牌：当前原型支持同点数组合，可带大王、小王、鹰作为挂牌。
- 比牌：必须同张数；先比最大挂牌/鹰，再比主体点数。

## 本地运行

```bash
npm install
npm start
```

浏览器打开：

```text
http://localhost:3000
```

手机测试：电脑和手机连接同一个 Wi-Fi，手机打开电脑局域网地址，例如：

```text
http://192.168.1.8:3000
```

## 部署

需要部署到支持 Node.js + WebSocket 的平台，例如 Render、Railway、Fly.io 或 VPS。部署后，用公网网址创建房间，再把邀请链接发给另外 5 个人。

Render 常用配置：

```text
Build Command: npm install
Start Command: npm start
```
